Growth Tool Release Note
=====
Growth Toolはプレイヤーの整地と共に成長していくツールです。<br />
Growth Toolはアイテムレベルを持ち、アイテムレベルの上昇と共に外見やエンチャントが変化します。<br />
また被ダメージや時間経過等のイベントにより、プレイヤーに対して話しかけてきます。<br />
